# US-003: User Logout

**Epic:** [Epic 1: User Authentication and Management](../epics/Epic-1.md)

**As a** logged-in user  
**I want to** logout  
**So that** my session is terminated

**Acceptance Criteria:**
- User can logout using their token
- Token is invalidated (if using token blacklist)
- Appropriate success response returned

**Technical Details:**
- Endpoint: `POST /api/v1/auth/logout`
- Headers: `Authorization: Bearer {token}`
- Response: `200 OK`
